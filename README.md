# 一個有GUI的MCPEDL月份排行爬蟲！
![image](https://user-images.githubusercontent.com/37009584/139248063-4dd337b4-12ff-43b5-ac30-6943db3fb5d6.png)

## 簡介
- 只是個小小的練習用專案
- 可以快速地爬取MCPEDL指定月份的熱門模組

## 功能與使用說明
- 1.選擇月份並輸入想取得的模組數量
- 2.按下確認，等待進度條跑完即可
- 3.打開生成的mods.html即可察看結果
